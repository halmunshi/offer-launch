# VSL Copy Framework

## VSL page
- Headline should interrupt pattern and speak directly to pain/desire.
- Subheadline clarifies audience and promise of the video.
- Write 5-7 curiosity bullets below the video.
- Use social proof transition when testimonials exist.
- Restate guarantee with confidence when available.
- CTA should be benefit-forward and immediate.

## Order page
- Reinforce the decision; do not re-sell from zero.
- Provide concise offer stack with clear value framing.
- Restate price and guarantee with no ambiguity.
- Include form-area support line above embed placeholder.
- CTA should match offer type: order or call booking.

## Thank You page
- Confirm action and celebrate briefly.
- Provide exact next steps with numbered instructions.
- Keep steps specific and practical.

## Upsell and downsell
- Upsell opener should create urgency around completing the journey.
- Show one logical extension offer and clear yes/no options.
- Downsell should acknowledge decline respectfully and offer lower-friction option.
- Keep downsell concise and direct.
